2. Process:

	2.1 To install all the dependencies required, please execute the following command in the terminal in the directory of 'cz4042_LiGuanlong_Heyuhao_P2/': 

		'''
		pip install -r requirements.txt
		'''
	2.2 CUDA Toolkit 10.0 and cuDNN v7.6.5 are required to run Part A Python scripts which utilize GPU for training models

	2.3 Assignment Part A:

		The Python scripts for part A are in directory 'cz4042_LiGuanlong_Heyuhao_P2/assignment1_question_a/'

		For question 1-3, run the following command (replace the X by the question number)

		'''
		python Part_A_Question_X.py
		'''

		The output of Python scripts include npy files and image files:
			The npy files are stored in 'cz4042_LiGuanlong_Heyuhao_P2/assignment1_question_a/others/npy/'
			The image files are stored in 'cz4042_LiGuanlong_Heyuhao_P2/assignment1_question_a/others/Figures'



	2.4 Assignment Part B:

		The Python scripts for part B are in directory 'cz4042_LiGuanlong_Heyuhao_P2/assignment1_question_b/'
